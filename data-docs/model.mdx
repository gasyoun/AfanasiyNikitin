# Модель данных

Двенадцать таблиц, связанных внешними ключами. Ядро — газеттир и итинерарий;
всё остальное ссылается на них и на реестр источников.

| Таблица | Что содержит | Строк |
|---|---|---|
| `sources.csv` | Реестр источников с колонкой `rights`; каждый `source_id` в других таблицах — внешний ключ сюда | 9 |
| `places.csv` | Газеттир **уникальных** мест (Бидар, Ормуз, Лар — по одному разу), с координатами и authority-ID | 26 |
| `itinerary.csv` | **Упорядоченные** 29 отрезков пути; `place_id` — ключ в `places` (повторы допустимы) | 29 |
| `legs.csv` | 28 переходов между соседними точками: расстояния (генерируется `tools/build_legs.py`) | 28 |
| `events.csv` | Узловые события пути: место, участники, тип, датировка | 12 |
| `people.csv` | Участники: кто помог, кто навредил, кто упомянут; сверка с VIAF/Wikidata | 15 |
| `edges.csv` | Рёбра сети людей (helped / harmed / neutral / indirect) | 17 |
| `fragments.csv` | 104 фрагмента текста (Ф.1–104): слой, жанр, период, место. **Только метаданные** — поле `quote` пусто (закрыто правами) | 104 |
| `calendar.csv` | Вычисленные Пасхи, Великие посты, Рамаданы 1467–1475 (генерируется `tools/computus.py`) | 37 |
| `editions.csv` | Издательские вехи текста, 1818–2026 | 9 |
| `citations.csv` | Оценка научного внимания по десятилетиям (`epistemic=model` — эвристика) | 12 |
| `trade.csv` | Торговые сведения: 4 текстовых факта + 19 модельных позиций | 23 |

## Соглашения

- **Кодировка** UTF-8 без BOM; CSV по RFC 4180.
- **`epistemic`** — пятизначный словарь типа утверждения:
  `text` (сказано в «Хожении») · `reconstruction` (реконструкция хронологии) ·
  `localization` (привязка к карте) · `model` (эвристическая оценка) ·
  `hypothesis` (гипотеза).
- **`certainty`** — отдельная ось надёжности: `certain` · `approx` · `disputed`.
- **`recon_status`** у мест и людей: `confirmed` / `candidate` / `none` /
  `collective` / `skip-region`; authority-ID записываются **только** для
  `confirmed` — неверный QID хуже пустого.
