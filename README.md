## Методология

### Датировка
Хрусталёв устанавливает **1467–1475** (вместо принятой 1468–1474) через астрономический анализ:
- Пасха 1469 г. (2 апреля) — в Ормузе ✓
- Совпадение Великого поста и Рамадана ~4 марта 1470 г. — «постился с бесерменами»
- Махашиваратри ~19 февраля 1471 г. — Шрисайлам
- «Филиппов пост» → Кафа, ноябрь 1474 г.

### Расстояния
Все км по **средневековым торговым путям XV в.**:
- Сухопутные — Великий шёлковый путь, персидские дороги, Деккан
- Морские — по лоциям Ахмада ибн Маджида (современник Афанасия)
- Волжский путь — по летописным данным о расстояниях

### Курс «золотого»
1 зол. ≈ венецианский дукат = **3.5 г чистого золота**
- По курсу золота $151.18/г (14 мая 2026): **1 зол. ≈ $529 ≈ ₽38 900**
- Обновлять курс раз в неделю в константах `GOLD_GRAM_USD` и `USD_RUB`
