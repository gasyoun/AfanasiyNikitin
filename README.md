## Что это такое

Первый в мире интерактивный атлас путешествия Афанасия Никитина — русского купца, который в 1467 году отплыл из Твери и через восемь лет, пройдя три моря, стал первым европейцем, оставившим письменный рассказ об Индии. Его рукопись едва не пропала — её нашли у мёртвого путника под Смоленском. Сейчас это один из главных памятников средневековой русской литературы.

Атлас основан на монографии историка Д.Г. Хрусталёва 2026 года — первой полной реконструкции первоначального состава «Хожения за три моря» с новой датировкой 1467–1475 гг.

**25 интерактивных визуализаций · Ваниллa JS · Без бэкенда · Работает офлайн**

## Сценарии просмотра

### 🕐 5 минут — «Быстрый обзор»
1. Откройте **Тайм-лапс** → нажмите ▶ → смотрите, как жёлтая точка проходит три моря за 2 минуты
2. Нажмите любую точку на карте → читайте цитату из «Хожения»
3. Заходите сюда же с телефона — атлас работает офлайн

### 🕐 20 минут — «Понять суть»
1. **Тайм-лапс** → смотрите маршрут
2. **Гантт-диаграмма** → поймите, почему Бидар занимает половину
3. **Три путешественника** → сравните с Марко Поло и Васко да Гамой (Ормуз — общая точка всех трёх!)
4. **Эмоциональная дуга** → видите три кризиса: ограбление на Волге, Джуннар, потеря Пасхи

### 🕐 45 минут — «Урок истории»
1. **Тайм-лапс** → маршрут на проекторе
2. **Пасхальный хронограф** → покажите главный аргумент новой датировки
3. **Параллель Гаван/Афанасий** → оба в Бидаре одновременно, оба купцы-в-изгнании
4. **Бестиарий** → первые русские описания слона, кокоса, попугая
5. **Маршрутник купца** → почему торговый план провалился

### 🕐 2 часа — «Глубокое погружение»
Пройдите все 6 групп последовательно:
1. Картографические (тайм-лапс → границы → море → Индия)
2. Хронологические (пасхи → гантт → скорость)
3. Текстовые (состав рукописи → языки → реконструкция)
4. Люди (параллель Гаван → граф → историография)
5. Экономика (маршрутник → жеребец → войны)
6. Контекст (бестиарий → три путешественника → рукописи)

### 📚 Для исследователя — «Источниковый анализ»
1. **Состав рукописи** → 104 фрагмента, 7 текстовых слоёв, 9 гипотетических тетрадей
2. **Реконструкция порядка** → как Мамырев перемешал листки при переписке
3. **Языковая тепловая карта** → различие публичного «Дорожника» и личного дневника
4. **Летопись войн** → записи Афанасия о военных кампаниях Бахманидов 1472–1473
5. **Три рукописи** → Летописная vs Троицкая: два утерянных листа и 580 знаков/страницу
6. **Календарный дрейф** → полный двойной православно-мусульманский календарь 1467–1475

## Методология

### Датировка (Хрусталёв 2026)
1467–1475 (вместо принятой 1468–1474) через астрономический анализ 8 Пасх:
- Пасха 2 апреля 1469 — в Ормузе ✓
- Совпадение Великого поста и Рамадана ~4 марта 1470 — «постился с бесерменами» ✓
- Махашиваратри ~19 февраля 1471 — Шрисайлам ✓
- «Шестая» Пасха 10 апреля 1474 — в Маскате ✓

### Расстояния
По средневековым торговым путям XV в.: шёлковый путь, персидские дороги, лоции Ахмада ибн Маджида.

### Курс «золотого»
1 зол. ≈ венецианский дукат = 3.5 г чистого золота. По курсу $151.18/г: **1 зол. ≈ $529 ≈ ₽38 900**
